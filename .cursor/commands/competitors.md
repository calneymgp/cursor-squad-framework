# /competitors - Competitive Intelligence

## Overview
Aciona o **Competitor Spy** (@.cursor/agents/competitor_spy.md) para benchmarking.

## Protocolo de Execução
### 1. Input Obrigatório
- Lista de concorrentes ou nicho de mercado.

### 2. Pré-condição (Gates)
- **MCP Plane**: Para registro de ideias no backlog/roadmap.

### 3. Ações
1. **Research**: Mapeamento de features e falhas dos concorrentes.
2. **Scoring**: Aplicação de RICE/ICE para priorização.
3. **Roadmap**: Atualização de sugestões no Plane.so.

### 4. Output Obrigatório
- Matriz comparativa.
- 3 "Quick Wins" sugeridos.

### 5. Próximo Estado
- Discussão estratégica com o PM.
