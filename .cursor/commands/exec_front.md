# /exec_front - Frontend Implementation (Unicorn Standard)

## Overview
Aciona o **Dev Frontend** (@.cursor/agents/dev_frontend.md) e o **UX Designer** (@.cursor/agents/ux_product_designer.md) para implementar a interface.

## Protocolo de Execução
### 1. Input Obrigatório
- Ticket ID ativo e validado via `/start`.

### 2. Skills Requeridas
- `@.cursor/skills/next-shadcn-ui/SKILL.md` (Padrões de UI)

### 2. Pré-condição (Gates)
- **DoD UX**: A régua visual deve estar definida.
- **MCP Plane**: Conexão ativa para comentários técnicos.

### 3. Ações
1. **UI**: Criação/modificação de componentes usando ShadCN + Framer Motion.
2. **Logic**: Integração com APIs/Supabase e tratamento de estados.
3. **UX Check**: Validação contra a regra `15-ux-ui.mdc`.

### 4. Output Obrigatório
- Código implementado seguindo os padrões do projeto.
- Evidência visual (descrição das animações e fluxos).

### 5. Próximo Estado
- Pronto para `/qa`.
