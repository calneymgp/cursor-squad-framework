# /copy - Copywriting Strategy

## Overview
Aciona o **Copywriter CTA** (@.cursor/agents/copywriter_cta.md).

## Protocolo de Execução
### 1. Input Obrigatório
- Contexto da feature ou campanha.

### 2. Ações
1. **Tone**: Definição do tom de voz.
2. **Headlines**: Criação de títulos de impacto.
3. **CTAs**: Variações de botões e links.

### 4. Output Obrigatório
- Lista de opções de copy prontas para uso.
