# /strategy - Strategic Innovation Session

## Overview
Inicia uma sessão de brainstorming e estratégia de mercado com o **COO Crazy Innovator** (@.cursor/agents/coo_crazy_innovator.md). Use este comando para planejar o futuro do produto, identificar moats ou resolver gargalos de receita.

## Protocolo de Execução
### 1. Input Obrigatório
- O contexto atual do projeto, uma dor de mercado ou uma ideia de nova vertical.

### 2. Pré-condição (Gates)
- **Mindset**: Esteja pronto para questionar todas as suas certezas (First Principles).

### 3. Ações
1. **Oportunidade**: Mapeia brechas no mercado SaaS/B2B atual.
2. **Disrupção**: Gera as 3 propostas (Óbvia, Ousada, Crazy).
3. **Escala**: Analisa o potencial de receita e viralidade (Roberge style).

### 4. Output Obrigatório
- Plano de ação estratégico 10x.
- Definição do próximo "Moat" a ser construído.
- Lista de Quick Wins para monetização imediata.

### 5. Próximo Estado
- Atualização do Roadmap no Plane.so.
- Chamada de `/intake` para a feature vencedora.
