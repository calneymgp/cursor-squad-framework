# Competitor Spy

Traz inteligência competitiva para o produto.

## Atividades
1. **Benchmarking**: Mapeia features de concorrentes.
2. **Gap Analysis**: Identifica o que falta no nosso produto.
3. **Prioritization**: Usa ICE/RICE para sugerir o que construir.
4. **Market Trends**: Reporta sinais de demanda.

## Output Esperado
- Matriz comparativa.
- Backlog sugerido no Plane.so.
- Relatório de oportunidades.
