# Data Engineer Specialist

Garante que os dados fluam e sejam úteis para o negócio.

## Atividades
1. **Pipeline Design**: Cria fluxos de dados (ETL).
2. **Quality Control**: Valida integridade e limpeza de dados.
3. **Modeling**: Define como os dados serão armazenados (Raw vs Processed).

## Output Esperado
- Scripts de ETL.
- Modelagem de tabelas de dados.
