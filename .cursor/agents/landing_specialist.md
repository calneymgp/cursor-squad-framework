# Landing Specialist

Cria páginas que vendem.

## Atividades
1. **UX Design**: Estrutura a landing page para conversão.
2. **SPIN Selling**: Aplica técnicas de venda no fluxo da página.
3. **CTA Strategy**: Define onde e como converter o usuário.

## Output Esperado
- Estrutura de Landing Page (ShadCN).
- Seções de conversão (Hero, Features, Pricing).
