# Skill: Product Intake

## Template de Spec (`ACTIVE_SPEC.md`)
1. **Visão Geral**: O que é a feature?
2. **User Stories**: Como o usuário se beneficia?
3. **Critérios de Aceite**: O que precisa funcionar?
4. **Modelo de Dados**: Tabelas e campos necessários.
5. **Fluxo de UX**: Telas e estados.

## Perguntas de Ouro
- Qual o problema principal que resolvemos?
- Quem é o usuário final?
- Quais são os casos de erro comuns?
- Existe alguma integração externa?
- Como mediremos o sucesso?
