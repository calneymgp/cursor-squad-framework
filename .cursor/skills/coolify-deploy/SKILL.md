# Skill: Coolify Deploy

## Preparação
- Dockerfile configurado corretamente.
- Variáveis de ambiente definidas no Coolify.
- Domínio apontado para o servidor.

## Deploy
- Webhook configurado para deploy automático.
- Verificação de logs pós-deploy.
- Cloudflare Zero Trust ativo para rotas sensíveis.
