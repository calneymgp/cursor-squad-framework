# Skill: Next & ShadCN UI

## Estrutura de Componentes
- `src/components/ui`: Componentes base do ShadCN.
- `src/components/features`: Componentes de negócio complexos.

## Padrões
- **Forms**: `react-hook-form` + `zod` + `shadcn/form`.
- **Modais**: `shadcn/dialog`.
- **Toasts**: `shadcn/sonner` para feedback.
- **Loading**: Use `Skeleton` para placeholders.
